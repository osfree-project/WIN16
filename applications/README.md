# Windows 3.0 Component Replacement Status

This project provides open‑source replacements for the classic applications that shipped with **Windows 3.0**.  
The goal is to recreate the original look, feel, and functionality.

Below is the current status of each component.

---

## Status Legend

| Badge | Meaning |
|-------|---------|
| ✅ **Complete** | Fully functional, tested, and ready for use. |
| 🚧 **In Progress** | Under active development; core features work, but some are missing. |
| ❌ **Not Started** | No work done yet; contributions welcome. |

---
